Geometric Algorithm Visualizations

The project will be an interactive web based educational tool to visualize a small set of geometric algorithms for computing the convex hull, line segment intersections, and polygon triangulation. Information describing each problem and algorithm will be provided before the interactive section. Additionally, the user interface will allow for the construction of input data as line segments, points, or polygons. The tool will be able to simultaneously highlight the geometric data currently being operated on and the high level algorithmic task being worked on. There will be the ability to step through each of the algorithms step by step or to automatically step through at various speeds. This should provide a visual link between the idea of the algorithm, and the visualized geometry of the problem.
